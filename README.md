Mini RPG Panel

Projeto simples de um painel de RPG feito com HTML, CSS e JavaScript.

## Funcionalidades
- Sistema de HP, Mana e XP
- Botão de batalha
- Uso de poções
- Sistema de level up

## Tecnologias usadas
- HTML
- CSS
- JavaScript

Projeto criado com foco em aprendizado prático e integração entre front-end e lógica.
